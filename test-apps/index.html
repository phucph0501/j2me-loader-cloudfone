J2ME Test Applications

Test applications designed for CloudFone platform and J2ME Loader

How to install:

  1. Download a JAR file by clicking the download button
  2. Open your J2ME Loader application
  3. Click "Install" or use left soft key
  4. Browse and select the downloaded JAR file
  5. Follow installation prompts
  6. Run the application and test keyboard functions
